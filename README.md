# order-dashboard

주문 관리 대시보드 (Order Dashboard)

주문 데이터를 기반으로
화면에 표시하기 적합한 UI용 데이터 구조를 만들고,  
그 흐름을 JavaScript와 React 중심으로 확장해가는 프로젝트입니다.

## 프로젝트 구조

- `orders.js`  
  주문 원본 데이터 정의

- `statusMap.js`  
  주문 상태 코드 -> 화면 표시용 한글 문구 매핑

- `buildOrdersUI.js`  
  주문 데이터를 UI 렌더링에 적합한 형태로 가공하는 로직

- `main.js`  
  전체 데이터 흐름을 연결하는 진입 파일

- `index.html`  
  화면 렌더링을 위한 HTML (추후 구현 예정)

## 현재 구현 내용

- 주문 상태 값을 기반으로 화면용 상태 문구 생성
- UI 렌더링을 고려한 주문 데이터 구조 설계
- 데이터 가공 로직을 함수 단위로 분리하여 관리
  (-> 실무적 상황 고려, 수정하기 쉽고 가독성 좋게 모듈화)

## 진행 예정

- 주문 리스트 화면 렌더링
- React 환경으로 확장
- 스타일링 및 사용자 인터랙션 추가
